# AI Trading Assistant

A real-time trading dashboard built with React and Vite that provides machine learning-driven signals, options setup recommendations, and market sentiment analysis.

## Features
- **AI Predictions**: Adaptive engine that analyzes technical indicators (RSI, MACD, EMA) to generate buy/sell signals.
- **Options Trading Setup**: Nifty 50 specialized options setups including CE/PE direction, Strike Price, Target, and Stop-Loss.
- **Market News Impact**: Live news scraper with Nifty 50 keyword sentiment analysis to filter high impact news.
- **Screener & Signals**: View live trading signals with success probabilities across both US and domestic (NSE) markets.
- **Mobile Responsive**: Perfectly optimized views for mobile devices.

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the dev server with `npm run dev`
4. Build for production with `npm run build`
